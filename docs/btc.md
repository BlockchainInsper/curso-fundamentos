# Bitcoin

Este seção é focada no estudo da histório e do funcionamento do Bitcoin. Por ter sido a primeira criptomoeda, o Bitcoin revolucionou o mercado financeiro e a maneira como se enxerga dinheiro. Por estes motivos, foi-se dedicado uma seção exclusiva para esta rede.
